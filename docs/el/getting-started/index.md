# Εγκατάσταση

## Προαπαιτούμενα

- Node.js 18+
- PostgreSQL 14+

## Βήματα Εγκατάστασης

1. Κλωνοποιήστε το repository
2. Εγκαταστήστε τις εξαρτήσεις
3. Ρυθμίστε τη βάση δεδομένων

## Παραδείγματα Κώδικα

```javascript
npm install
npm run dev
```
